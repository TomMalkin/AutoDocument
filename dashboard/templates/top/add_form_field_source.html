How to use

Add a Field for a user to fill out.

Form Fields are always the first part of a Workflow that is run.

The value input by the user can be accessed using the Name of the Field, while the Label of the field is what the user will see on the form.

For example, if the name is client_name, and the label is Client Name, then the text that the user input can be accessed in your templates using 'client_name'.
