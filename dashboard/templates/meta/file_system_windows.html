Why add File Systems?

AutoDocument can read and write documents directly from a given filesystem.

The output path can be templated, so if you are generating a document for each client, then a template like this is possible!:

{% raw %} N:\business_folder\clients\{{ client_id }}\[{{ invoice_id }}] Invoice.pdf {% endraw %}

Which would create files:

  • N:\business_folder\clients\123\[9] Invoice.pdf
  • N:\business_folder\clients\234\[10] Invoice.pdf
  • N:\business_folder\clients\345\[11] Invoice.pdf
  • etc...
